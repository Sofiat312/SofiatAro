My Resume!

Excel

Microsoft Excel is a powerful spreadsheet application widely used for data entry, analysis, visualization, and automation. It enables users to perform tasks ranging from basic calculations to complex data modeling.

SQL

SQL is the foundation of modern relational databases. It allows users to manage and query data efficiently and is a critical skill for developers, data analysts, and database administrators.

Tableau

Tableau is a powerful, user-friendly data visualization and business intelligence tool developed to help people understand their data. It allows users to create interactive, shareable dashboards and perform data analysis from a wide variety of sources.

Power Bi

Power BI is a business analytics tool developed by Microsoft that allows users to visualize data, share insights, and make informed business decisions. It transforms raw data from multiple sources into interactive dashboards and reports.

Python

Python is a versatile, high-level programming language known for its simplicity, readability, and wide range of applications in data analysis, web development, automation, machine learning, and more.

R-Programming

R is a powerful programming language and environment primarily used for statistical computing, data analysis, and data visualization. It is highly favored by statisticians, data scientists, and researchers for its extensive libraries and statistical capabilities.
